# api-toolkit

Lightweight REST API framework with middleware support

## Installation

```bash
pip install api-toolkit
```

## Quick Start

```python
from api_toolkit import Api
client = Api()
client.run()
```

## Features

- High performance and memory efficient
- Comprehensive error handling
- Well tested with full coverage
- MIT licensed
